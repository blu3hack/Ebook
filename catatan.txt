===== langkah-langkah git =====
1. buat repository di github seperti biasa
2. masuk ke directory folder lokal lalu ketik : git init
2. git add . (menambahkan semua file project ke staging area)
4. git status (cek dulu statusnya)
5. git commit -m "tambahkan pesan keterangan"
6. git remote add origin "https://github.com/blu3hack/Telegram_bot.git" disini akan diminta untuk login ke akun github kita
7. git branch -M main
9. git push -u origin main

====== langkah-langkh instalasi project =============
1. composer create-project laravel/laravel:^11 .
2. composer require laravel/breeze --dev && php artisan breeze:install react


======= tambahkan font poppins =======
1. Buka resources/css/app.css dan tambahkan di paling atas:
@import url('https://fonts.googleapis.com/css2?family=Poppins:wght@100;200;300;400;500;600;700;800;900&display=swap');
2. Buka tailwind.config.js: set poppins sebagai font default

====== menambahkan sqlite ============
1. pertama tambahkan variable baru di .env => SQLITE_DATABASE=database/database.sqlite
2. di config/database.php ubah di bagian sqlite => 'database' => env('SQLITE_DATABASE', database_path('database.sqlite')),
3. buat migration database seperti biasa.
4. php artisan migrate --database=sqlite (untuk migrate ke sqlite)
5. cek dengan tinker (php artisan tinker)
6. DB::connection('sqlite')->select("SELECT name FROM sqlite_master WHERE type='table';"); (untuk melihat daftar table)
