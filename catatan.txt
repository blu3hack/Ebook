===== langkah-langkah git =====
1. buat repository di github seperti biasa
2. masuk ke directory folder lokal lalu ketik : git init
2. git add . (menambahkan semua file project ke staging area)
4. git status (cek dulu statusnya)
5. git commit -m "tambahkan pesan keterangan"
6. git remote add origin "https://github.com/blu3hack/Telegram_bot.git" disini akan diminta untuk login ke akun github kita
7. git branch -M main
9. git push -u origin main

====== langkah-langkh instalasi project =============
1. composer create-project laravel/laravel:^11 .
2. composer require laravel/breeze --dev && php artisan breeze:install react
